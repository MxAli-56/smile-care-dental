Smile Care Dental Clinic - Official Website
This is the official web platform for Smile Care Dental Clinic located in Gulshan-e-Iqbal, Karachi. The site is designed to provide patients with clinical information and features a live integration with the FlexiBot AI Concierge.

🚀 Features:

1. AI Concierge: Real-time dental assistant powered by Mistral AI.
2. Responsive Design: Fully optimized for mobile, tablet, and desktop.
3. Doctor Schedules: Live-updated availability for General Dentistry, Orthodontics, and Oral Surgery.
4. High Performance: Hosted on Vercel for instant loading speeds.

🛠️ Tech Stack
. Frontend: HTML5, CSS3, JavaScript.
. AI Integration: FlexiBot Embed Script.
. Hosting: Vercel.

AI Integration Details
The chatbot is injected via a custom script tag connected to the FlexiBot Render backend.
